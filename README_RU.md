# CineCLI

Инструмент командной строки для поиска и просмотра фильмов напрямую с Rutracker в терминале через mpv и webtorrent.

## Возможности

- Поиск торрентов на Rutracker из командной строки
- Прямой просмотр фильмов в mpv без скачивания
- Быстрый доступ к магнет-ссылкам
- Интерактивный режим для удобного выбора
- Конфигурация через файл настроек
- Поддержка аутентификации через cookies браузера

## Требования

- Python 3.10 и выше
- Медиаплеер mpv
- Инструмент webtorrent CLI
- Node.js (для установки webtorrent)

## Установка

### 1. Установка Python зависимостей

```bash
pip install cinecli-ru
```

### 2. Установка системных зависимостей

#### macOS

```bash
brew install mpv
npm install -g webtorrent-cli
```

#### Linux (Ubuntu/Debian)

```bash
sudo apt-get install mpv
sudo apt-get install npm
sudo npm install -g webtorrent-cli
```

#### Linux (Fedora/RHEL)

```bash
sudo dnf install mpv
sudo dnf install npm
sudo npm install -g webtorrent-cli
```

### 3. Настройка учетных данных

Создайте файл конфигурации в `~/.config/cinecli-ru/config.toml`:

```bash
mkdir -p ~/.config/cinecli-ru
```

Отредактируйте `~/.config/cinecli-ru/config.toml`:

```toml
[rutracker]
username = "ваш_логин"
password = "ваш_пароль"

[settings]
default_action = "magnet"   # "magnet" или "torrent"
default_limit  = 20
```

#### Аутентификация через cookie браузера (рекомендуется)

Для большей надежности вы можете использовать cookie браузера вместо хранения пароля:

1. Откройте rutracker.org в браузере и авторизуйтесь
2. Откройте Developer Tools (F12)
3. Перейдите в Application (Приложение) → Cookies → rutracker.org
4. Найдите cookie `bb_session` и скопируйте его значение (начинается с "0-")
5. Добавьте в файл конфигурации:

```toml
[rutracker]
username = "ваш_логин"
password = "ваш_пароль"
cookie = "bb_session=0-XXXXXXXXXXXXX"

[settings]
default_action = "magnet"
default_limit  = 20
```

Инструмент будет использовать cookie вместо пароля, если он указан.

## Использование

### Поиск фильма

```bash
cinecli-ru search матрица
cinecli-ru search "темный рыцарь" --limit 10
```

### Смотреть фильм напрямую

Автоматически выбирает лучший торрент и запускает просмотр в mpv:

```bash
cinecli-ru watch матрица
cinecli-ru watch начало --limit 10
```

### Получить детали торрента и выбрать действие

Загрузить информацию о конкретном торренте и выбрать действие:

```bash
cinecli-ru get 12345
```

Доступные действия:
- `stream` - Смотреть напрямую в mpv через webtorrent
- `magnet` - Открыть магнет-ссылку в торрент-клиенте
- `page` - Открыть страницу торрента в браузере
- `skip` - Пропустить и выйти

### Интерактивный режим

Запустить интерактивную сессию:

```bash
cinecli-ru interactive
```

Режим позволит вам:
1. Ввести поисковый запрос
2. Просмотреть результаты поиска
3. Выбрать, автоматически выбрать лучший торрент (по количеству сидов и размеру)
4. Выбрать между просмотром, магнет-ссылкой, страницей или выходом

## Опции

### Команды search и watch

- `--limit N` или `-n N` - Максимальное количество результатов поиска (по умолчанию: 20)

Примеры:
```bash
cinecli-ru search "бойцовский клуб" --limit 5
cinecli-ru watch "побег из шоушенка" -n 15
```

## Качество потокового просмотра

Инструмент автоматически выбирает торренты, оптимизированные для потокового просмотра:

- Предпочитает разрешение 1080p
- Приоритизирует форматы WebRip и BD-Rip (меньший размер, быстрый старт)
- Избегает больших remux файлов
- Выбирает торренты с наибольшим количеством сидов

Примечание: Очень большие файлы (>10 ГБ) могут долго буферизоваться. Для лучшего опыта просмотра выбирайте качество WEBRip или BDRip (2-4 ГБ).

## Решение проблем

### Ошибка аутентификации

Если вы видите сообщение "Rutracker login failed":

1. Проверьте корректность логина и пароля
2. Попробуйте использовать вместо этого cookie из браузера (см. раздел Конфигурация)
3. Убедитесь, что Rutracker доступен
4. Получите cookie из браузера, следуя инструкциям выше

### webtorrent не найден

Установите webtorrent глобально:

```bash
npm install -g webtorrent-cli
```

Проверьте установку:

```bash
webtorrent --version
```

### mpv не найден

Установите mpv для вашей системы:

- macOS: `brew install mpv`
- Ubuntu: `sudo apt-get install mpv`
- Fedora: `sudo dnf install mpv`

### Медленный просмотр

- Проверьте скорость интернета
- Попробуйте торрент с большим количеством сидов
- Используйте `cinecli-ru get <id>` и выберите меньший файл (WebRip/BDRip)
- Избегайте больших remux файлов для потокового просмотра

### Ошибка API

Если вы видите предупреждение об недоступности API, API Rutracker временно недоступен. Инструмент автоматически переключится на парсинг веб-интерфейса.

## Детали конфигурационного файла

### Значения по умолчанию

```toml
[rutracker]
username = ""           # Обязательно
password = ""           # Обязательно
cookie = ""            # Опционально, приоритет выше чем пароль

[settings]
default_action = "magnet"   # Действие когда --action не указан
default_limit = 20          # Ограничение по умолчанию для результатов поиска
```

### Расположение конфигурации

- Linux/macOS: `~/.config/cinecli-ru/config.toml`
- Windows: `%APPDATA%\cinecli-ru\config.toml`
